# Currency Converter
A simple web application for converting currencies in real time.

## Overview
Currency Converter is a lightweight web app built with HTML, CSS, and JavaScript. It lets users convert any amount from one currency to another, using up-to-date exchange rates. The interface is straightforward and responsive, suitable for both desktop and mobile use.

## Features
- Select source and target currencies
- Instantly calculate conversions as you type
- Responsive design and user-friendly controls

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
